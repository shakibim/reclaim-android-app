## About the Project

**ReClaim** is an Android app built with **Java**, **XML**, and **Firebase** that helps users report and recover lost items.  
It features **user authentication**, the ability to **post lost or found items**, **view posts in a timeline**, **comment on posts**, and manage personal activity through a **history section** where users can **edit or delete posts**.  
The app also includes a **profile settings** page for user customization.
